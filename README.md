# volumeVulkan
- Inital setup with MSYS2 clang64
- Required: `cmake` `ninja`, but cmake needs to be in Windows as to get correct Backends
# Build
- `mkdir` a `build` folder
- `cmake .. -G "Ninja" -D CMAKE_C_COMPILER=clang -D CMAKE_CXX_COMPILER=clang++ -D BUILD_SHARED_LIBS=OFF`
- Run Ninja
